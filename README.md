
# 🚖 Cab Booking Analytics Dashboard

A complete Data Analytics project built using **SQL, Power BI, and Excel** to analyze cab booking operations. This project focuses on extracting meaningful business insights from a large-scale booking dataset through SQL analysis and an interactive Power BI dashboard.

 📌 Project Overview

The objective of this project is to analyze cab booking data to understand customer behavior, revenue trends, booking patterns, cancellation rates, and operational performance. The project demonstrates end-to-end data analysis, from querying raw data using SQL to creating an interactive dashboard in Power BI.

🛠️ Tech Stack

- SQL (MySQL)
- Microsoft Power BI
- Microsoft Excel
- GitHub

📂 Dataset Information

The dataset contains approximately 70,000 cab booking records with the following information:

- Booking ID
- Customer ID
- Driver ID
- Booking Date & Time
- Pickup City
- Drop City
- Vehicle Type
- Ride Distance
- Ride Duration
- Booking Status
- Base Fare
- Surge Multiplier
- Total Fare
- Payment Method
- Customer Rating
- Driver Rating
- Cancellation Reason
- Promo Code
- Weather
- Traffic Level
